Use your onePlace Events in Wordpress

== Description ==

If you have onePlace and want to use its data in WordPress for an Online Store,
or Show Events this plugin is for you.

* connect to your oneplace server and fetch calendar data
* show your events
* sell your event tickets
* including elementor widgets

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enable the settings you want in "WP PLC Event" in the OnePlace Menu in Settings Panel

== Frequently Asked Questions ==
